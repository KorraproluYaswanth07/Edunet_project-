Simple Banking System

The Simple Banking System is a basic C program that simulates essential banking operations, such as creating accounts, depositing and withdrawing money, and checking balances. It demonstrates the use of structures, file handling, and basic input/output operations. Users can create accounts, make deposits and withdrawals, and check account details. This project effectively showcases fundamental programming concepts and provides a foundation for more complex banking applications.

Sudoku

The Sudoku Game in C++ allows users to solve the puzzle by filling a 9x9 grid with numbers 1 to 9 in each row, column, and 3x3 subgrid without repetition. It includes a predefined puzzle with validation for user inputs to avoid duplicates and features an automatic solver using backtracking for solutions, showcasing arrays, recursion, and user interaction in C++.
